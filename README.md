# **GWE 2026 Data Science Challenge: Hypertension Risk Prediction**

**Tema Utama**: Leveraging Data for Societal and Industrial Impact

**Sub-Tema**: Risk Prediction

Tema yang kami angkat dalam Data Science adalah Risk Prediction yang menggunakan model machine learning Random Forest. Proyek ini bertujuan untuk membangun sebuah model end-to-end yang dapat memprediksi risiko hipertensi berdasarkan metrik kesehatan dan faktor gaya hidup pasien. 

**Latar Belakang & Tujuan**:
 Hipertensi merupakan salah satu faktor risiko utama untuk penyakit kardiovaskular. Melalui proyek ini, kami menganalisis hubungan antara gaya hidup (seperti konsumsi garam, tingkat stres, dan durasi tidur) dengan riwayat tekanan darah. Output dari proyek ini adalah sebuah aplikasi interaktif berbasis Streamlit yang dapat digunakan untuk melakukan risk prediction* secara real-time.

**Dataset**:
 Dataset yang digunakan dalam proyek ini berisi metrik kesehatan sintetis/dunia nyata pasien, mencakup variabel numerik (BMI, Salt Intake) dan kategorikal (Smoking Status, Family History).

**Sumber Dataset**: https://www.kaggle.com/datasets/miadul/hypertension-risk-prediction-dataset

**Metodologi & Lingkup Pengerjaan**:
Proses Data Science dalam proyek ini dilakukan secara end-to-end, mencakup:

**Data Preparation**: Penanganan missing values (seperti pada kolom pengobatan) dan feature engineering (pengelompokan kategori BMI).

**Exploratory Data Analysis (EDA)**: Analisis distribusi dan korelasi antar variabel gaya hidup terhadap target prediksi.

**Machine Learning Modeling**: Menggunakan Ensemble Method Random Forest Classifier yang ideal untuk memproses data non-linear dan menghindari overfitting.

**Deployment**: Model di-deploy menggunakan antarmuka interaktif Streamlit.

**Penggunaan AI selama pembuatan**:
- Machine learning modelling.
- Data Encoding & Correlation Analysis.
- Distribusi BMI & Salt Intake.
- Streamlit application development.
- Dashboard visualization implementation.
- Debugging and troubleshooting during development.
- Deployment guidance using GitHub and Streamlit Community Cloud.
